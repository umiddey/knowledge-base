# SkyWater 130nm PDK Documentation

The SkyWater PDK documentation is the standard reference for the open-source Sky130 process stack.

## What it covers
- Process design kit structure and install guidance
- Device models, libraries, and design collateral
- Open-source ASIC flow dependencies
- Interfacing with OpenLane and OpenROAD-based flows

## Why it matters
- It is the foundational process reference for open-source ASIC experimentation
- It anchors the `open-source-asic-flow` concept
- It gives practical context for building and tapeout in a real foundry process
