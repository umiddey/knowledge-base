# RISC-V Unprivileged ISA Manual

This is the canonical ISA reference for the base user-level RISC-V instruction set.

## What it covers
- Base instruction formats and behavior
- Integer, control-flow, and memory access semantics
- Extension model for multiply, atomics, floating point, and vector support
- The architectural baseline for custom CPU and accelerator designs

## Why it matters
- It is the main primary source for `risc-v-processor-design`
- It explains how control CPUs and embedded cores are specified
- It is essential background for AI chips that embed RISC-V control logic
