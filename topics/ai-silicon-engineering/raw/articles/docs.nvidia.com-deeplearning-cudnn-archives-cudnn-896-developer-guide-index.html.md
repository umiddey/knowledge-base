# NVIDIA cuDNN Developer Guide

cuDNN is the core NVIDIA library for accelerated deep neural network primitives.

## What it covers
- Convolution, normalization, activation, and recurrent-layer primitives
- Tensor format and datatype support
- Backend graph APIs and engine selection
- Performance, algorithm choice, and determinism tradeoffs
- Integration points with broader NVIDIA deep learning stacks

## Why it matters
- It is the main low-level library behind many GPU training and inference stacks
- It explains the performance constraints compilers and frameworks need to respect
- It fills a gap between CUDA programming and high-level model execution
