# StableHLO

StableHLO is the portable MLIR dialect and interchange layer used across the OpenXLA ecosystem.

## What it covers
- A stable representation for machine learning computations
- Compatibility between framework frontends and compiler backends
- A bridge between high-level model graphs and lower-level hardware codegen

## Why it matters
- It reduces churn in compiler pipelines that target many devices
- It is a key part of the modern XLA/OpenXLA stack
- It helps explain how model portability works across accelerators
