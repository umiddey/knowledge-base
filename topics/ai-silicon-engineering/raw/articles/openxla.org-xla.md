# OpenXLA / XLA

XLA is the compiler stack used to optimize machine learning workloads for accelerators such as GPUs and TPUs.

## What it covers
- High-level optimization of ML graphs
- Lowering from framework IRs into device-specific code
- TPU and GPU execution backends
- Relationship to StableHLO and the wider OpenXLA ecosystem

## Why it matters
- It is one of the main compiler stacks in modern ML infrastructure
- It directly connects JAX and TensorFlow-style workloads to accelerator hardware
- It is a canonical source for the `ai-compilers` concept
