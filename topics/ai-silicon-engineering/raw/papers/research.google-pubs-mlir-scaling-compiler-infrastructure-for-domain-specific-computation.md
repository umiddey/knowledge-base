# MLIR: Scaling Compiler Infrastructure for Domain Specific Computation

This is the core MLIR paper and a canonical reference for modern multi-level compiler infrastructure.

## Core ideas
- Reusable compiler infrastructure for heterogeneous and domain-specific computation
- Multiple abstraction levels, from graph-level operations to low-level code generation
- Extensible dialect system for building new compiler pipelines
- Better support for hardware acceleration and cross-domain code reuse

## Why it matters
- It is the main primary source behind MLIR's design philosophy
- It is essential background for accelerators, compilers, and codegen work
- It fills a major gap in the `ai-compilers` reading list
