# Eyeriss v2: A Flexible Accelerator for Emerging Deep Neural Networks on Mobile Devices

Eyeriss v2 is a canonical dataflow-accelerator paper that extends the original Eyeriss line to compact and sparse neural networks.

## Core ideas
- Flexible on-chip network for varying layer shapes and reuse patterns
- Direct support for sparse weights and activations in compressed form
- Systematic analysis methodology for accelerator performance limits
- Emphasis on mobile and resource-constrained deployment targets

## Why it matters
- It is a strong reference for accelerator dataflow design
- It shows how sparsity changes both architecture and software assumptions
- It belongs in the deep technical layer of `ai-accelerator-design`
